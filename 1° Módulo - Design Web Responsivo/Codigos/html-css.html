            Lista não ordenada:
Uma lista não ordenada é definida pelo elemento "ul" e não possui tipo de marcadores. Essa lista não ordenada de links virou um menu com botões e alguns efeitos interessantes do css.

Efeitos usados:

            nav li{
                display: inline-block;
            }
            nav a{
                display: inline-block;
                padding: 5px 1em;
                background-color: lightgray;
                color: darkslategray;
            }
            nav a:hover{
                background-color: deepskyblue;
            }
            nav ul{
                padding-left: 0;
            }

Posicionamento

Existem três esquemas de posicionamento:

Normal flow: elementos de nivel de paragrafo em formatação de bloco e caixas inline.

Floats: são caixas flutuantes. Não se usa o atributo position e sim o float

Absolute positioning: fixa a imagem nas posições. exemplo feito com a imagem do css
