Aula 3.03

Declaração de variáveis

Os nomes de variáveis são case sensitive. Exemplos:

        var nome;
        var Nome;
        var NOME;

Essas três variáveis são diferentes, por causa das letras maiúsculas e minúsculas.

//comentario de linha

/* comentario de varias linhas */

Se não usar var ou let para declarar variaveis, as variaveis serão globais.

Porém, ao colocar "use script" não é possível declarar uma variável deixando de usar var ou let.

A variável pode ter seu tipo mudado durante a execução do script, podendo ser boolean, num momento, number em outra e string um pouco depois. Basta atribuir valores destes tipos durante a execução.

Aula 3.07

Funções

Uma função é declarada pela palavra reservada function. Exemplo: 

        function soma (x,y) {
            console.log(arguments);
            switch(arguments.length){
                case 0: return 0;
                case 1: return x;
                case 2: return y;
            }
        }
        
        console.log(soma(5,2));
        console.log(soma(3));
        console.log(soma());
        console.log(soma(1,2,3,4,5));

Não muda muito se comparado as funções do java. MAs existe outra forma de declarar funções que é por meio de uma expressão. O que é uma expressão? Algo que retorna um valor. Exemplos: 

        
        console.log(3);
        console.log(1+58*3);
        console.log(a>5*b);
        console.log( {nome: "João", idade: 35});

Todos esses comandos acima são expressões.

Uma função é um objeto em javascript.

Aula3.11 - Classes

Classes - Syntatical sugar - Significa que é um agrado para quem está habituado com outras linguagens

        class Poligono {
            constructor(largura, altura){
                this.largura = largura;
                this.altura = altura;
            }
            get area(){
                return this.altura*this.largura;
            }
        }
        
        
        class Ponto{
            constructor(x,y){
                this.x = x;
                this.y = y;
            }
            static distancia(a,b){
                const dx = a.x-b.x;
                const dx = a.y-b.y;
                return Math.sqrt(dx*dx + dy*dy);
            }
            toString(){
                return '(' + this.x+','+this.y+')';
            }
            const p1 = new Ponto(5,5);
            const p2 = new Ponto(10,10);
            console.log(Ponto.distancia(p1,p2));
        }
