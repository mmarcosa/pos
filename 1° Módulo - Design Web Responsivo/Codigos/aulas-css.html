Cascade Style Sheets

Define um conjunto de regras de estilo que permitem a apresentação do conteúdo Estrutura: 
                seletor{
                    propriedade1: valor1
                    propriedade2: valor2
                    propriedade3: valor3
                }
Exemplo prático de alteração de títuloo h1: 
                h1{
                    font-family: Arial;
                    font-size: 16pt;
                    font-weight: bold;
                }

Vantagens da formatação com CSS

  • Formatação de vários elementos;
  • Reaproveitamento da formatação em várias páginas;
  • Facilicade de manutenção do conteúdo e da fomatação;
  • Aplicação de diferentes estilos;
Pode-se incorporar o css por três maneiras:

Link: use a tag link e faça o import do arquivo .css

Style: use a tag style e faça um css dentro do formulário;

No próprio html: nãoi use isso ..

Aplicar a mais de um seletor: 
        p, ul{  
            background-color: yellow;
        }
        
        h2{
            background-color: green;
        }
Aplicar a um filho de um seletor: 
        li p{
            background-color: gray;
        }
        
        header{
            font-family: sans-serif;
        }
        
        blockquote{
            background-color: red;
        }
        
        footer{
            font-size: smaller;
        }
Aplicar a todos: 
        *{
           background-color: blue;
        }
É possível pegar pseudo elementos, por exemplo a primeira letra de um parágrafo: 
        p::first-letter{
            font-size: 30pt;    
        }

Seletor de identificador

atribua um id a um seletor e faça o estilo desse identificador. Exemplo:

            #username{
                font-size: 30pt; 
            }

Formatação de texto

Formatação de fontes

            body{
                font-family: calibri, Arial, sans-serif;
                font-weight: bold;     --> basicamente indica se é negrito
            }
O exemplo acima fala que caso o navegador não encontre a calibri, arial pode ser usada.

Modelos de caixa

Ao redor do conteúdo há uma borda. Vamos mexer com essas propriedades (margem interna, externa e propriedades da borda) para vermos mais sobre o assunto.

Exemplos de formatação de margens de paragrafo

        
        p{
            text-align: justify;
            line-height: :1.25;
            text-indent: 2em;
            background-color: darkmagenta;
            border: 2px solid chartreuse;
        }
        
        #p1{
            margin-bottom: 0, 1em;   --> primeiro pra cima, segundo pra baixo
        }
        
        #p2{
            margin-top: 0, 1em, 2em; --> primeiro pra cima, segundo para os lados terceiro pra baixo
        }
        
        #p2{
            margin-top: 0, 1em, 3em, 2em; --> segue o sentido horário
        }

a borda pode ser especidicada através de trÊs plropkriedades:

        p{

            border-color: chartreuse;
        }
